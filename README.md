# Oricham Start LP C

オリシャン事業導入サービス CパターンLPです。

## Stack

- Next.js
- React
- Tailwind CSS
- Static export対応

## Vercel Environment Variables

Vercel側で以下を設定してください。

```env
NEXT_PUBLIC_GA_ID=G-3D4NC4BTG1
```

任意で後から追加できます。

```env
NEXT_PUBLIC_CLARITY_ID=
NEXT_PUBLIC_LINE_URL=
NEXT_PUBLIC_GOOGLE_FORM_URL=
NEXT_PUBLIC_DOWNLOAD_URL=
```

`NEXT_PUBLIC_GA_ID` が未設定でもエラーにならない構成です。

## Commands

```bash
npm install
npm run dev
npm run build
```
